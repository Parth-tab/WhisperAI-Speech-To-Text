# WhisperAI-Speech-To-Text-
WhisperAI: Open-source, offline voice dictation for Windows. I built a multi-threaded AI pipeline combining Whisper, Silero VAD, and local LLMs that runs completely on CPU. It features an always-on-top frameless UI, auto-edits filler words, and pastes context-aware text directly into your IDE. 100% local. 0% cloud.
